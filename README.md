# Nekoverse - Cat Adoption Landing Page

Nekoverse is a sleek, modern landing page designed for a fictional company that connects people with cute cats for adoption. This project was developed as part of **[The Odin Project's](https://www.theodinproject.com/)** curriculum to practice and master CSS Flexbox and responsive web design.

## 🐾 Project Overview

The Nekoverse landing page provides a comprehensive overview of the adoption service, featuring a hero section to grab attention, a gallery of featured cats, inspiring quotes, and a clear call-to-action (CTA).

## ✨ Features

- **Hero Section**: A high-impact introduction with a call-to-action "Sign Up" button and a placeholder for a hero image.
- **Responsive Navigation**: Clear, clean links to Home, Cats, Quotes, and Contact sections without underlines.
- **Featured Cats Gallery**: A 4-column layout showcasing different cat breeds.
- **Testimonial Quote**: A stylishly formatted quote section featuring a background highlight and italicized text.
- **Call to Action**: A prominent "Call Us" section designed to convert visitors into adopters.
- **Modern Design System**: Consistent use of a professional color palette (`#1f2937` for darks, `#3882F6` for blues) and curated typography.

## 🛠️ Built With

- **HTML5**: Semantic elements for structure.
- **CSS3**: Custom layouts powered primarily by **Flexbox**.

## 🚀 Getting Started

Since this is a static website, you can view it by simply cloning the repository and opening the `index.html` file in any modern web browser.

1.  **Clone the repo**:
    ```bash
    git clone https://github.com/Everwander0327/My-Landing-Page.git
    ```
2.  **Open the project**:
    Navigate to the project directory and open `index.html` in your browser.
